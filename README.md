# Java Codes

A comprehensive collection of Java programs covering Core Java, OOP, Collections, Generics, Design Patterns, and Problem Solving.

## 📚 Topics Covered

### Object Oriented Programming (OOP)
- `OOPcode.java` - Object Oriented Programming concepts
- `ObjandClass.java` - Classes and Objects
- `ObjectClassEg.java` - Object class methods
- `EgOFInheritance.java` - Inheritance examples
- `EgofAbstract.java` - Abstract classes
- `AbstractClass.java` - Abstract class implementation
- `InterfacesDeepDive.java` - Interface deep dive
- `InterfaceChangesInAfterJava8.java` - Java 8+ interface features
- `DiamondProblem.java` & `DiamondProblemDemo.java` - Diamond problem resolution
- `AnonymousClass.java` - Anonymous inner classes

### Advanced Java Concepts
- `StaticKeyword.java` & `StaticClassDemo.java` - Static members and classes
- `FinalAndStaticDemo.java` - Final and static keywords
- `ImmutableClassEg.java` & `ImmutableClassEg2.java` - Immutable class design
- `EnumExample.java` - Enumeration usage
- `BuilderDemo.java` - Builder design pattern

### Collections Framework
- `ArrayListDemo.java` - ArrayList implementation
- `CollectionEg1.java` - Collection basics
- `SetMapEg1.java` - Set and Map interfaces
- `IteratorEg.java` - Iterator usage

### Generics
- `GenericsEg.java` - Generic classes and methods
- `GenericsEg3.java` - Generic examples
- `GenericsFullDemo.java` - Comprehensive generics
- `wildcardsGenerics1.java` - Wildcards in generics

### Strings & StringBuilder
- `StringsExamples.java` - String manipulation
- `StringBuilderDemo.java` - StringBuilder basics
- `StringBuilderEg.java` & `StringBuilderEg3.java` - StringBuilder examples

### Arrays & Type Conversion
- `Arrays.java` - Array operations
- `TypeConversion.java` & `TypeConversion2.java` - Type casting
- `AutoboxingAndInboxing.java` - Autoboxing/unboxing

### Problem Solving & Algorithms
- `AllPatternsPrinter.java` - Pattern printing
- `DutchNationalFlag.java` - DNF algorithm
- `Primenumbers.java` - Prime number operations
- `Banking.java` - Banking system implementation

### Java I/O & Utilities
- `IOexamples.java` - Input/Output operations
- `ScannerAllMethodsDemo.java` - Scanner class methods
- `MathConstUse.java` - Mathematical constants
- `random.java` - Random number generation

### Additional Examples
- `Demo.java` & `demo5.java` - Demonstration programs
- `CompilerTest.java` - Compiler behavior
- `Functionsnew.java` - Function implementations
- `pojoclass.java` - POJO class example
- `TypeConversion2.java` - Advanced type conversion
